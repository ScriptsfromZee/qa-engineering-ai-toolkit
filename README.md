# QA Tool Kit

AI-powered prompt toolkit built for QA Engineers.

Created by Derek Chizogam.

This repository contains production-ready prompts designed to help QA Engineers move faster with:

- Test case generation
- Bug report writing
- Regression planning
- API test suite building
- Developer pushback responses
- Flaky test debugging

The goal is simple:

Reduce time spent on repetitive documentation so QA Engineers can focus on actual testing.

---

# What's Inside

## Included Prompt Systems

### 1. Test Case Generator
Generate structured, high-coverage test cases from requirements.

### 2. Bug Report Writer
Turn messy notes into professional bug reports developers can act on.

### 3. Developer Pushback Response Generator
Draft evidence-based responses when bugs are dismissed incorrectly.

### 4. Regression Test Prioritiser
Build risk-based regression plans for releases.

### 5. Flaky Test Debugger
Diagnose unstable automated tests systematically.

### 6. API Test Suite Builder
Generate structured API test suites for REST and GraphQL systems.

# How To Use

## Option 1 — ChatGPT Custom Instructions

Go to:

Settings → Personalization → Custom Instructions

Paste the system prompt into:

"How should ChatGPT respond?"

---

## Option 2 — Claude Projects

Create a new project and paste the system prompt into:

Project Instructions

---

## Option 3 — Any Other LLM

Start a new chat and paste:

```text
For this session, follow these instructions:
```

Then paste the system prompt.
